¿Qué es Python?

Python es un lenguaje de programación interpretado, de alto nivel y propósito general, creado por Guido van Rossum en 1991. Su filosofía prioriza la legibilidad del código.

Interpretado Se ejecuta línea por línea sin necesidad de compilación previa.
Tipado dinámico El tipo de variable se infiere en tiempo de ejecución.
Multi-paradigma Soporta POO, programación funcional e imperativa.
Gran ecosistema Más de 400,000 paquetes disponibles en PyPI.
02

Variables y asignación

En Python no es necesario declarar el tipo de una variable. La asignación se realiza con el operador =.

ejemplo.py
# Asignación simple
nombre = "Ana"
edad   = 28
altura = 1.65
activo = True

# Asignación múltiple
x, y, z = 1, 2, 3

# Ver tipo de variable
print(type(nombre))  # <class 'str'>

Estructuras de control

Python utiliza la indentación para delimitar bloques de código. Los condicionales y ciclos son la base del flujo de ejecución.

control.py
# Condicional
if edad >= 18:
    print("Mayor de edad")
elif edad >= 13:
    print("Adolescente")
else:
    print("Menor de edad")

# Ciclo for
for i in range(5):
    print(i)

# Ciclo while
contador = 0
while contador < 3:
    contador += 1
05

Funciones

Las funciones permiten encapsular lógica reutilizable. Se definen con la palabra clave def.

funciones.py
# Función básica
def saludar(nombre):
    return f"Hola, {nombre}!"

# Parámetros con valor por defecto
def potencia(base, exponente=2):
    return base ** exponente

# Función lambda (anónima)
doble = lambda x: x * 2

print(saludar("María"))  # Hola, María!
print(potencia(3))        # 9

Listas y diccionarios

Las listas y los diccionarios son las estructuras de datos más utilizadas en Python.

colecciones.py
# Lista
frutas = ["manzana", "pera", "mango"]
frutas.append("kiwi")
print(frutas[0])    # manzana

# Comprensión de lista
cuadrados = [x**2 for x in range(5)]

# Diccionario
persona = {
    "nombre": "Luis",
    "edad":   30,
    "ciudad": "CDMX"
}
print(persona["nombre"])  # Luis
07

Clases y objetos

Python soporta programación orientada a objetos. Las clases agrupan datos (atributos) y comportamiento (métodos).

clases.py
class Animal:
    def __init__(self, nombre, especie):
        self.nombre  = nombre
        self.especie = especie

    def hablar(self):
        return f"{self.nombre} hace un sonido"

# Herencia
class Perro(Animal):
    def hablar(self):
        return f"{self.nombre} dice: ¡Guau!"

rex = Perro("Rex", "Canino")
print(rex.hablar())  # Rex dice: ¡Guau!
08

Módulos e importaciones

Los módulos permiten organizar y reutilizar código. Python incluye una extensa biblioteca estándar lista para usar.

modulos.py
# Importar módulo completo
import math
print(math.sqrt(16))   # 4.0

# Importar función específica
from datetime import date
print(date.today())

# Alias
import random as rnd
print(rnd.randint(1, 100))
