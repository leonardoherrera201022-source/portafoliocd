¿Qué es HTML?

concepto.html

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. No es un lenguaje de programación — es un lenguaje de marcado: describe la estructura y el contenido.

💡 dato clave HTML5 es la versión actual desde 2014. Trajo etiquetas semánticas, soporte nativo para audio/video y muchas mejoras.
🛠 herramienta recomendada Usa VS Code + extensión "Live Server" para ver tus cambios en tiempo real.
🏗️
02

Estructura básica

index.html — plantilla mínima 
<!DOCTYPE html>                   <!-- Declara HTML5 -->
<html lang="es">
  <head>
    <meta charset="UTF-8">           <!-- Codificación de caracteres -->
    <meta name="viewport" content="width=device-width">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
  </body>
</html>
📊 ÁRBOL DE ELEMENTOS (DOM)
<html> — Raíz del documento
<head> — Metadatos (no visible en el navegador)
<title>, <meta>, <link>, <script>
<body> — Contenido visible de la página
<header>, <main>, <section>, <footer> …

Imágenes

etiqueta img 
<!-- Imagen desde URL -->
<img
  src="https://ejemplo.com/foto.jpg"
  alt="Descripción de la imagen"
  width="400"
  height="300"
>

<!-- Imagen local -->
<img src="imagenes/logo.png" alt="Logo">
AtributoDescripción¿Obligatorio?
src Ruta o URL de la imagen Sí
alt Texto alternativo (accesibilidad + SEO) Sí
width Ancho en píxeles Opcional
height Alto en píxeles Opcional
vista previa — cómo se ve en el navegador

alt="Paisaje de montañas con lago"

♿ accesibilidad El atributo alt es leído por lectores de pantalla. Si la imagen es decorativa, usa alt="" (vacío).

Hipervínculos

etiqueta a (anchor) 
<!-- Enlace externo -->
<a href="https://www.google.com" target="_blank">
  Ir a Google
</a>

<!-- Enlace interno -->
<a href="contacto.html">Contacto</a>

<!-- Enlace a sección del mismo documento -->
<a href="#inicio">Volver arriba</a>

<!-- Enlace de correo -->
<a href="mailto:hola@ejemplo.com">Enviar correo</a>

<!-- Imagen como enlace -->
<a href="https://ejemplo.com">
  <img src="logo.png" alt="Logo">
</a>
Ir a Google · Contacto · hola@ejemplo.com
⚠️ target="_blank" Al usar target="_blank" añade siempre rel="noopener noreferrer" por seguridad.

Formularios

tipos de input comunes 
<form action="/enviar" method="POST">

  <!-- Texto -->
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" placeholder="Tu nombre">

  <input type="email" name="email" placeholder="correo@ejemplo.com">

  <!-- Número -->
  <input type="number" min="1" max="100">

  <!-- Contraseña -->
  <input type="password" name="pass">

  <!-- Fecha -->
  <input type="date" name="fecha">

  <!-- Checkbox -->
  <input type="checkbox" name="acepto"> Acepto términos

  <!-- Radio -->

  <input type="radio" name="sexo" value="f"> Femenino

  <!-- Área de texto -->
  <textarea rows="4" cols="40"></textarea>

  <!-- Botón de envío -->
  <button type="submit">Enviar</button>

</form>

HTML Semántico

📖 concepto Las etiquetas semánticas de HTML5 describen el significado del contenido, no solo su apariencia. Mejoran el SEO y la accesibilidad.
estructura semántica típica 
<body>
  <header>    <!-- Encabezado del sitio -->
    <nav>Menú de navegación</nav>
  </header>

  <main>      <!-- Contenido principal -->
    <section>  <!-- Sección temática -->
      <article> <!-- Contenido independiente -->
        <h2>Noticia</h2>
        <p>Texto del artículo…</p>
      </article>
    </section>
    <aside>    <!-- Barra lateral -->
      <p>Contenido relacionado</p>
    </aside>
  </main>

  <footer>    <!-- Pie de página -->
    <p>© 2025 Mi Sitio</p>
  </footer>
</body>
<header> — Logo, menú
<main>
<section>
<article>
<aside>
sidebar
<footer> — Copyright
