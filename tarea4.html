01

Selectores

Los selectores determinan a qué elementos HTML se aplican los estilos.

Vista previa
p { } — Tipo
.clase { } — Clase
#id { } — ID
a:hover { } — Pseudo
[type="text"] — Atributo
/* Selector de tipo */
p { color: blue; }

/* Selector de clase */
.titulo { font-size: 24px; }

/* Selector de ID */
#encabezado { background: black; }

/* Combinado: hijo directo */
nav > ul > li { list-style: none; }

/* Selector de atributo */
input[type="email"] { border: 2px solid teal; }

07

Transiciones

Las transiciones animan suavemente el cambio de un estado CSS a otro.

Vista previa — pasa el cursor sobre los botones
.boton {
  background:  #e8ff47;
  color:       #000;
  padding:     12px 24px;

  /* propiedad | duración | función | retraso */
  transition: background .3s ease, transform .3s ease;
}

.boton:hover {
  background:  #ff6b4a;
  transform:   scale(1.05);
}

/* Funciones de easing disponibles */
/* ease | linear | ease-in | ease-out | cubic-bezier() */
08

Animaciones

Con @keyframes puedes definir animaciones complejas de múltiples pasos sin JavaScript.

Vista previa — animaciones en tiempo real
float
spin
pulse
slide-in
/* 1. Define los fotogramas clave */
@keyframes flotar {
  0%, 100% { transform: translateY(0); }
  50%       { transform: translateY(-12px); }
}

/* 2. Aplica la animación al elemento */
.elemento {
  animation-name:            flotar;
  animation-duration:        2s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;

  /* Forma corta */
  animation: flotar 2s ease-in-out infinite;
}
09

Pseudo-clases

Las pseudo-clases aplican estilos basados en el estado o posición de un elemento.

Vista previa — interactúa con la lista
  • :first-child — primer elemento
  • :nth-child(even) — fondo alterno
  • :hover — cambia al pasar el cursor
  • :nth-child(even) — fondo alterno
  • :last-child — último elemento
li:first-child    { border-top: 2px solid #ff6b4a; }
li:last-child     { border-bottom: 2px solid #4af0c4; }
li:nth-child(even){ background: rgba(255,255,255,.04); }

li:hover {
  color:        #e8ff47;
  padding-left: 28px; /* combina con transition */
}

/* Pseudo-elementos */
li:hover::before {
  content: '▶';
  position: absolute;
  left: 10px;
}

Variables CSS

Las Custom Properties (variables) permiten reutilizar valores y crear temas dinámicos.

Vista previa — paleta definida con variables
--accent
--accent2
--accent3
--bg
--card
/* Define en :root para acceso global */
:root {
  --color-primario: #e8ff47;
  --color-fondo:    #0d0d0f;
  --espaciado-base: 16px;
  --radio-borde:    4px;
}

/* Usa con var() */
.boton {
  background:    var(--color-primario);
  padding:       var(--espaciado-base);
  border-radius: var(--radio-borde);
}

/* var(propiedad, valor-por-defecto) */
color: var(--color-texto, #fff);

Diseño Responsivo

Con Media Queries adaptas el diseño a diferentes tamaños de pantalla o dispositivos.

Vista previa — grid auto-adaptable
Móvil
Tablet
Desktop
Wide
/* Breakpoints comunes */
@media (max-width: 480px)  { /* Móvil */ }
@media (max-width: 768px)  { /* Tablet */ }
@media (min-width: 1024px) { /* Desktop */ }

/* Ejemplo práctico */
.grid { grid-template-columns: repeat(3, 1fr); }

@media (max-width: 768px) {
  .grid { grid-template-columns: 1fr; }
}

/* Grid responsivo sin media queries */
.auto-grid {
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}
